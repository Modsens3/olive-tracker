# Olive Tracker

Μια εφαρμογή για την καταγραφή ελιών και την εκτίμηση της παραγωγής.

## Χαρακτηριστικά
- Καταγραφή τοποθεσίας ελιών με GPS
- Αποθήκευση ποικιλίας, υγείας και εκτίμησης παραγωγής
- Εξαγωγή δεδομένων σε JSON
- Λειτουργία Offline (PWA)

## Εγκατάσταση
Η εφαρμογή είναι διαθέσιμη online.
Για τοπική ανάπτυξη:
1. `npm install`
2. `npm run dev`
